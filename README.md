# BGN → EUR Change Calculator

A simple web app to help small shop sellers in Bulgaria handle the transition from **BGN to EUR**.

The app calculates:
- Total purchase in EUR
- Payment made in **EUR, BGN, or both**
- Change returned in **EUR**, with automatic fallback to **BGN** if there isn’t enough EUR cash available

## Tech Stack

- React + TypeScript
- Vite
- CSS (no UI framework)
- Deployed on Railway
